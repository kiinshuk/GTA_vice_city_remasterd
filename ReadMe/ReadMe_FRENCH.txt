 1) CONFIGURATIONS SYSTEME 
________________________

Grand Theft Auto: Vice City requiert Microsoft DirectX 9.0.

Les systèmes d'exploitation suivants sont compatibles avec le jeu :

	Windows 98
	Windows 98 Deuxième Edition
	Windows Millennium
	Windows 2000 Professionnel (Station de travail) plus Service Pack 3
	Windows XP (Familial et Professionnel) plus Service Pack 1

Les systèmes d'exploitation suivants ne sont pas compatibles :

	Windows 95 (toutes versions)
	Windows NT (toutes versions)

Configuration minimale
	
	Processeur Intel Pentium III 800 MHz ou AMD Athlon 800 MHz  
		ou processeur Intel Celeron 1,2GHz ou AMD Duron 1,2 GHz 
	256 Mo de mémoire vive (RAM)
	Lecteur CD/DVD-ROM 8x
	915 Mo d'espace libre sur le disque dur 
		(+ 635 Mo si la carte graphique N'est PAS compatible avec la compression de textures DirectX)
	Carte graphique 32 Mo avec pilotes compatibles DirectX 9.0  
		("GeForce" ou mieux)
	Carte son avec pilotes compatibles DirectX 9.0 
	Clavier
	Souris

Configuration recommandée

	Processeur Intel Pentium IV ou AMD Athlon XP 
	512 (ou plus) Mo de mémoire vive (RAM)
	Lecteur CD/DVD-ROM 16x
	1,55 Go d'espace libre sur le disque dur 
		(+ 635 Mo si la carte graphique N'est PAS compatible avec la compression de textures DirectX)
	Carte graphique 32 Mo avec pilotes compatibles DirectX 9.0 
		("GeForce 3"/"Radeon 8500" ou mieux, compatible avec la compression de textures DirectX)
	Carte son avec son surround et pilotes compatibles DirectX 9.0 
	Manette de jeu (Port USB ou série)
	Clavier
	Souris

 2) INSTALLATION
_________________

Veuillez insérer le disque 1 de Grand Theft Auto: Vice City (disque "INSTALLATION") dans 
le lecteur de CD/DVD-ROM. Si la fonction d'exécution automatique est activée, l'écran de configuration se lance automatiquement. Si ce n'est pas le cas, utilisez l'explorateur de Windows pour parcourir le disque 1 de Grand Theft Auto: Vice City, puis lancez le fichier setup.exe manuellement.

Compte utilisateur sous Windows 2000/XP :
Grand Theft Auto: Vice City doit être installé et lancé pour la première fois avec un compte
ayant les privilèges ADMINISTRATEUR. Lors du premier lancement du jeu, le programme décidera s'il faut utiliser ou non la compression de textures DirectX et si nécessaire, décompressera
les textures du jeu sur votre disque dur. Pour plus d'informations sur les comptes ADMINISTRATEUR, 
veuillez consulter votre documentation Windows 2000/XP.

Surface d'encombrement:
Grand Theft Auto: Vice City requiert environ 915 Mo d'espace libre sur votre disque dur si vous choisissez de ne pas installer les stations de radio sur votre disque dur. En revanche, si vous copiez les stations de radio sur votre disque dur, vous devez alors disposer de 1,55 Go d'espace libre. Si la carte graphique de votre ordinateur n'est pas compatible avec la compression de textures DirectX, vous devez disposer de 635 Mo d'espace libre supplémentaire.

Stations de radio :
Pendant l'installation, nous vous proposons de copier sur votre disque dur les fichiers des stations de radio qui se trouvent sur le disque "JEU". Sur certaines machines, ceci peut améliorer 
les performances. Toutefois, pour les machines moins performantes, nous recommandons de laisser les stations de radio sur un périphérique IDE différent. Quoiqu'il en soit, vous aurez toujours besoin du disque "JEU" inséré dans votre lecteur, que les stations de radio soient copiées sur le disque ou non.

Sauvegardes :
Grand Theft Auto: Vice City crée un répertoire "GTA VICE CITY USER FILES" dans le répertoire
"Mes Documents" de l'utilisateur. C'est à cet endroit que sont conservés vos sauvegardes, les fichiers de configuration ainsi que les statistiques de jeu. Ce répertoire a été choisi pour permettre aux utilisateurs Windows 2000 ou Windows XP d'avoir leurs propres sauvegardes, configurations et statistiques.

Désinstallation :
Quand Grand Theft Auto: Vice City est désinstallé, les dossiers "MP3", "SKINS" 
et "GTA VICE CITY USER FILES" ne sont pas désinstallés. C'est à l'utilisateur de les effacer manuellement.

 3)INFORMATIONS GENERALES
________________________

DirectX :
Grand Theft Auto: Vice City requiert Microsoft DirectX 9.0 pour fonctionner. En conséquence,
nous vous recommandons d'utiliser des pilotes sonores et graphiques compatibles avec DirectX 9.0. 

Compression de textures DirectX :
Lorsque cela est possible, Grand Theft Auto: Vice City utilise la compression de textures DirectX. 
Si la carte graphique n'est pas compatible avec cette fonction, le jeu nécessitera 635 Mo d'espace libre supplémentaire pour créer les textures non compressées (en plus des 915 Mo de l'installation minimale ou des 1,55 Go de l'installation recommandée).

Windows XP :
Nous vous recommandons d'installer les pilotes du fabricant pour Windows XP (c'est-à-dire les pilotes qui affichent les commandes d'utilisateur liées aux changements de paramètres matériels), à la place des pilotes par défaut installés par Microsoft lors de la configuration de la machine. 

Mémoire :
Grand Theft Auto: Vice City nécessite au moins 128 Mo de mémoire pour fonctionner correctement. Dans la plupart des cas, sur un ordinateur doté de 128 Mo de RAM, le jeu utilisera la mémoire virtuelle. Pour des performances optimales, nous vous recommandons au moins 256 Mo de RAM.

Tâches de fond :
Grand Theft Auto: Vice City a été conçu pour exploiter au maximum les ressources fournies par la machine sur laquelle il a été installé, c'est pourquoi nous vous recommandons de fermer toutes les autres applications fonctionnant en arrière-plan avant de lancer Grand Theft Auto: Vice City. Si vous utilisez des émulateurs de CD, ces programmes seront automatiquement fermés au lancement du jeu. 

Mémoire graphique :
Grand Theft Auto: Vice City requiert une carte graphique avec 32 Mo de mémoire minimum. Le jeu ne fonctionnera pas sur une machine ne disposant pas d'au moins 32 Mo de mémoire graphique. 

Processeurs Intel Celeron et AMD Duron :
Ces processeurs sont plus lents que les Pentium III et AMD Athlon équivalents. Nous vous conseillons donc une vitesse minimale de 1,2 GHz pour ces processeurs.

 4) CONFIGURATION
__________________

Gestionnaire audio :
Lors du premier lancement de Grand Theft Auto: Vice City, le programme détecte automatiquement le gestionnaire 
audio le plus approprié pour votre carte son. Vous pouvez aussi en sélectionner un autre manuellement.
Les gestionnaires qui ne sont pas compatibles avec votre carte son ne seront pas accessibles.

Volume MP3 à fond :
Ce paramètre vous permet d'attribuer de la puissance supplémentaire au son de vos fichiers MP3 diffusés en arrière plan. Ceci est accessible seulement si le jeu détecte des fichiers MP3 dans le répertoire réservé aux MP3.

Modelage acoustique dynamique:
Désactivez cette option dans les options sonores pour gagner un peu en performances.

Résolution :
Sélectionnez une faible résolution graphique pour que Grand Theft Auto: Vice City soit plus rapide sur les machines moins puissantes. Par ailleurs, n'hésitez pas à choisir des couleurs 16 bits, au lieu de couleurs 32 bits pour améliorer les performances de jeu.

Limite d'images :
Nous vous conseillons de ne pas désactiver cette option sur l'écran de configuration de 
Grand Theft Auto: Vice City car cela risquerait de rendre le jeu "instable".

VSYNC :
Nous vous recommandons de vérifier dans les options avancées de votre carte graphique que l'option VSYNC est activée (dans Panneau de configuration/Propriétés de affichage).

 5) SUPPLEMENTS DU JEU PC
_______________________
Skins (apparences) :
Les skins de Tommy peuvent être modifiés en ouvrant un fichier de skins avec un éditeur d'images.
Les fichiers de skins doivent être des bitmaps 24 bits décompressés, au format 256 * 256.
Un fichier d'exemple est fourni dans le répertoire "SKINS", il se nomme "TEXTURE_GUIDE.JPG".
Lorsqu'ils sont appliqués, les skins modifient l'apparence de Tommy dans les rues de la ville et dans le jeu uniquement (pas dans les cinématiques).

Fichiers MP3 des stations de radio :
vous avez la possibilité de diffuser vos propres MP3 au lieu des fichiers sonores des stations de radio d'origine.
Pour cela, il vous suffit de copier vos fichiers MP3 dans le répertoire "Grand Theft Auto Vice City/MP3", puis d'utiliser la touche assignée au changement de station pour passer en revue vos MP3. Activez l'option "Volume MP3 à fond" pour améliorer le son de vos fichiers MP3 dans le jeu.

Sauvegarde des statistiques :
Pendant le jeu, vous pouvez faire ressortir la liste détaillée de ce que vous avez accompli à partir du menu de statistiques. Dans ce menu, appuyez sur la touche "S" pour créer un fichier "STATS.TXT" et un fichier "STATS.HTML" dans le répertoire "GTA VICE CITY USER FILES" de l'utilisateur actuel.

Mode standard/classique : 
Le mode standard vous permet de contrôler Tommy avec le clavier et la souris alors que le mode classique est identique au système de contrôle dans Grand Theft Auto: Vice City sur PlayStation2 avec une manette de jeu.

Conduite avec la souris :
En mode standard, une option vous permet d'utiliser la souris pour diriger les véhicules.
Cette option n'est pas disponible en mode classique.

Ralentis :
A tout moment, vous pouvez visionner les 30 dernières secondes de jeu.

	F1 - Rejouer les 30 dernières secondes de jeu. 
	F2 - Sauvegarder les 30 dernières secondes de jeu sous forme de fichier. 
	F3 - Afficher le dernier fichier de ralenti.

 6) PROBLEMES CONNUS
_________________

Installation Windows 98 :
Sur certaines machines équipées de Windows 98, il peut y avoir des problèmes avec la version du programme d'installation (Installshield) que nous utilisons. Ceci peut être dû à un fichier "OLEAUT32.DLL" invalide. La plupart du temps, ce fichier est une version que celle installée lors du processus de configuration de la machine. Ce problème se manifeste habituellement par un message d'erreur lorsque vous redémarrez votre ordinateur après l'installation de DirectX 9.0. 
Ce message d'erreur est du type :- 

Erreur numéro : 0x80070725
Description : version incompatible du RPC stub

Pour résoudre ce problème, désinstallez Grand Theft Auto: Vice City (si les raccourcis vers la "barre Démarrer" et l'option "Ajout/suppression de programmes" ne fonctionnent pas, désinstallez en lançant le processus de configuration à partir du disque d'installation de Grand Theft Auto: Vice City. Il s'agit là d'un symptôme type de cette erreur DCOM). Une fois que vous avez désinstallé Grand Theft Auto: Vice City, procédez comme suit pour corriger votre installation DCOM. Vous pourrez alors installer Grand Theft Auto: Vice City correctement. Nous sommes désolés de ne pas pouvoir redistribuer ce fichier, mais le contrat de licence de Microsoft pour ce fichier nous l'interdit.

Instructions pour remédier à ce problème (de l'assistance de Microsoft Installshield)

1) Téléchargez sur votre Bureau, mais ne lancez pas, le fichier Distributed Component Object Model 
   (DCOM). 
   Windows 98 : http://support.installshield.com/kb/files/Q105810/dcom98.exe

2) Sélectionnez Démarrer > Arrêter 

3) Sélectionnez Redémarrer en mode MS-DOS. 

4) Dans l'invite de commande c:\WINDOWS, saisissez les commandes suivantes sur des lignes séparées comme indiqué ci-après. Windows sera automatiquement relancé. 

	CD c:\windows\system
	rename oleaut32.dll oleaut32.old
	exit

5) Ignorez le message d'erreur : "A required .DLL file, C:\windows\system\oleaut32.dll, was not found" (fichier .DLL, C:\windows\system\oleaut32.dll, introuvable). Ce message s'affiche parce que le fichier a été renommé à l'étape 4. 

6) Connectez-vous comme à votre habitude. 

7) Double-cliquez sur le fichier que vous avez téléchargé sur votre Bureau à l'étape 1. Cliquez sur Oui à l'invite qui s'affiche. Le fichier dcom.exe installe un nouveau fichier oleaut32.dll. 

8) Après l'installation du DCOM, sélectionnez Démarrer > Redémarrer.

Pour plus d'informations sur ce problème et sur DCOM :-

http://support.installshield.com/kb/view.asp?pcode=ALL&articleid=Q105810#addinfo
http://www.microsoft.com/com/dcom/dcom98/dcom1_3.asp


Cartes graphiques S3 Pro Savage, Savage 4 et Savage 2000 :
Nous avons rencontré des problèmes de textures avec ces cartes graphiques réglées sur des couleurs 16 bits. Nous vous suggérons donc d'utiliser ces cartes avec des couleurs 32 bits.

Cartes graphique 3dfx Voodoo 5 :
Alors que Grand Theft Auto: Vice City fonctionne avec les cartes graphiques Voodoo 5, l'ancienneté des pilotes entraîne des petits bugs graphiques qui ne peuvent pas être corrigés, ce qui fait que le jeu n'est pas compatible avec cette carte ni avec les 3dfx précédentes.

Matrox surround :
Nous n'avons pas prévu la compatibilité du jeu avec le mode Matrox surround ni avec le mode 3 monitor. Nous vous conseillons donc d'utiliser le mode écran unique avec les cartes Matrox Parhelia.

Cartes graphiques Matrox G550 et G400 :
Nous avons rencontré de petits problèmes au niveau des graphismes à distance pendant le jeu et pendant la cinématique d'introduction.

Cartes graphiques ATI Rage 128 :
Petits problèmes graphiques pendant la cinématique d'introduction.

Cartes vidéo SIS 315 :
Petits problèmes graphiques pendant la cinématique d'introduction.

Carte-mère avec son intégré :
Nous nous sommes rendus compte que certains périphériques de son intégrés à la carte-mère n'ont pas de mémoire tampon pour le son et plantent avec Grand Theft Auto: Vice City. En lançant le fichier "DXDIAG" de Microsoft et en effectuant les tests sonores, un message d'erreur vous informe que la mémoire tampon matériel n'est pas compatible. Une invite vous propose d'utiliser la mémoire logiciel. Si vous acceptez, vous pouvez alors jouer à Grand Theft Auto: Vice City.

Défragmentation du disque dur :
S'il vous arrive de voir des ralentissements courts mais récurrents pendant que vous conduisez, il se peut que Grand Theft Auto: Vice City rencontre des problèmes pour retrouver les données suffisamment rapidement sur votre disque dur.
Nous vous conseillons alors de défragmenter régulièrement votre disque dur pour bénéficier de performances optimales.

Changements de matériel/réinitialisation :
Le simple fait de changer votre carte graphique ou votre carte son peut contribuer à ce que Grand Theft Auto: Vice City ne reconnaisse pas votre matériel. Si vous changez de matériel et si vous voulez éviter de désinstaller et réinstaller Grand Theft Auto: Vice City, procédez comme suit :
Carte graphique :- 
	- supprimez du répertoire "Grand Theft Auto Vice City/models", les fichiers
	  "TXD.DIR" et "TXD.IMG" (s'ils existent)
	- supprimez du répertoire "Grand Theft Auto Vice City/data", le fichier
	  "CAPS.DAT"
	- supprimez du répertoire "Documents and Settings/Current logged-in User/
	  Mes documents/GTA Vice City User Files", le fichier "GTA_VC.SET".
Carte son :- 
	- supprimez du répertoire "Documents and Settings/Current logged-in User/
	  Mes documents/GTA Vice City User Files", le fichier "GTA_VC.SET".
